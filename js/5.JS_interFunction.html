자바스크립트 내장함수

인코딩과 디코딩에 관련된 내장함수

1. escape : 영문 알파벳, 숫자, 일부 특수문자(@, *, -, _, ., /)를 제외한 모든 문자 인코딩
2. encodeURI : escape에서 인터넷 주소에 사용되는 일부 특수문자(:, ;, /, =, ?, &)는 변환하지 않음.
3. encodeURIComponent : 알파벳과 숫자를 제외한 모든 문자를 인코딩(UTF-8 방식)
4. unescape : escape로 인코딩된 문자를 디코딩함.
5. decodeURI : encodeURI로 인코딩된 문자를 디코딩함
6. decodeURIComponent : encodeURIComponent로 인코딩된 문자를 디코딩함.
