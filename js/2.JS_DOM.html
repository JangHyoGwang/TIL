Javascript를 이용한 HTML 태그 접근

자바스크립트를 이용하여 HTML 태그를 동적으로 처리하기 위해서는 DOM객체의 메소드를 호출하여 접근해야함. 브라우저를 실행하는 순간 DOM객체 형성되고 객체안에 모든 정보가 저장(트리형태).

DOM객체 메소드

1. getElementById("아이디명") : 태그의 id 속성값을 이용해서 태그 요소 객체정보를 가져옴(1개)
2. getElementByName("이름명") : 태그 name 속성값을 이용해서 태그 요소의 객체 정보를 배열에 담아 가져옴(여러개 존재할 수 있음.)
3. getElementByTagName("태그명") : 태그명을 이용해서 태그들의 객체 정보를 배열에 담아 가져옴(여러개 존재할 수 있음.)
